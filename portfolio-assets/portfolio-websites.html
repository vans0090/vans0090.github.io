MY WEBSITES

This was one of our project where we had to copy a provided mockup by using grid or flexbox for the entire layout. I chose grid as it is quite convenient and flexible. The sources and images were given and we had to mock them in the same way. The main technique that I used was using grid-row property to keep it in the background.

D'Cruze Cake Poppies was my final foundation project assignment in first semester. We were given Purple as the primary color, which I combined with yellow color to reflect joy of the sweets. A brief introduction, products that my imaginary shop serves, a second banner for the main quote and final form and footer was a perfect combination of the sugar poppies candy shop and bakery.

This was my first project in second semester where we had to redesign a website provided by the professor. I prepared a simple elegant layout with a combination of dark and light colors. Using flexbox for image and text in about section and providing 80% width to the website I completed this project. The main objective of the project was to use high specificity to change the element's styling.
